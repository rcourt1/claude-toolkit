# Prompt: GitHub Pages portfolio website

Build a modern GitHub Pages portfolio website that is fully static, professional, and easy to edit.

## Technical requirements

- Files: `index.html`, `style.css`, `script.js` only
- No frameworks, build tools, or external dependencies (standard fonts/icons are fine)
- All text content editable in `index.html`
- All styling in `style.css`
- All interactivity in `script.js`
- Responsive and suitable for GitHub Pages hosting

## Sections

Single-page layout with:

1. Hero
2. About / Summary
3. Skills or Services
4. Projects / Work
5. Experience / Background
6. Contact

Also include: profile image, CV download button, navigation menu, smooth scrolling, mobile layout, subtle animations, and an optional light/dark mode toggle.

## Design

- Modern, clean, and professional
- Easy to read, suitable for recruiters, clients, or collaborators
- Avoid: flashy effects, cluttered layouts, anything that looks dated or generic
